# 제주여행 웹앱 — 배포 & 카카오 지도 켜기 (5분)

폰에서 **현재 위치(GPS) + 카카오 지도 + 길찾기**를 쓰려면 https 주소로 올려야 합니다.
아래 순서만 따라 하면 됩니다.

---

## 1단계 · 카카오 JavaScript 키 넣기 ✅ (완료됨)

`index.html`에 JavaScript 키(`cfafcd33…`)를 **이미 넣어뒀습니다.** 로컬에서 지도가 뜨는 것도 확인했어요.
확인만 하려면 아래 줄이 이렇게 돼 있으면 OK:

```js
var KAKAO_JS_KEY = "cfafcd3304aec9f2c2f56d21a6163940";
```

혹시 지도가 회색이면 카카오맵 제품이 꺼져 있는 것:
[앱 설정 > 카카오맵] → **사용 ON** (또는 [제품 설정]에서 지도/로컬 활성화)

---

## 2단계 · GitHub Pages로 올리기

1. https://github.com → 로그인 → 우측 상단 **+** → **New repository**
   - 이름 예: `jeju-trip` · **Public** · Create
2. 만들어진 저장소 화면에서 **uploading an existing file** 클릭
3. 이 폴더의 **`index.html`** 파일을 드래그해서 올리고 **Commit changes**
4. 상단 **Settings** → 왼쪽 **Pages**
   - Source: **Deploy from a branch**
   - Branch: **main / (root)** → **Save**
5. 1~2분 뒤 새로고침하면 주소가 나옵니다:
   `https://<본인아이디>.github.io/jeju-trip/`

> 이 주소가 **폰에서 열 최종 주소**입니다. 홈 화면에 추가해두면 앱처럼 써요.

---

## 3단계 · 카카오에 도메인 등록 (지도가 안 뜨면 이것 때문)

1. 카카오 개발자 → 내 애플리케이션 → **앱 설정 > 플랫폼**
2. **Web 플랫폼 등록** → 사이트 도메인에 추가:
   ```
   https://<본인아이디>.github.io
   ```
3. 저장 후 웹앱 새로고침 → 지도가 뜹니다.

---

## 확인 체크리스트
- [ ] JavaScript 키를 index.html에 넣었다 (REST 키 아님)
- [ ] 카카오맵 사용 ON
- [ ] GitHub Pages 주소가 나온다
- [ ] 카카오 플랫폼에 github.io 도메인 등록
- [ ] 폰에서 열고 **📍 내 위치** 눌러 권한 허용

---

## 안 될 때
| 증상 | 원인 / 해결 |
|---|---|
| 지도 자리에 주황 안내문만 | JS 키 안 넣음 → 1단계 |
| 지도가 회색 빈칸 | 도메인 미등록 → 3단계 |
| "내 위치" 눌러도 안 됨 | https 주소로 안 열었거나 권한 거부. github.io 주소로 열고 권한 허용 |
| 맛집 영업시간 | 카드에 "미확인" 표시된 곳은 **가기 전 전화 확인** (네이버 조사 기준, 시간은 자주 바뀜) |
